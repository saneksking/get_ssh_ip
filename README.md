**Get SSH IP** - Telegram Bot for get a command for connect to your SSH server
---

## Help:

- Clone project
- Create file `config.json` in your project and enter:
```json
{
    "token": "your telegram token",
    "admins": [
        "chat_id1",
        "chat_id1"
    ]
}
```

---

## Run:

`python app.py`
